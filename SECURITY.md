# Security Policy

## Supported versions

The current `main` branch is maintained.

## Reporting a vulnerability

Please do not open a public issue for sensitive security reports.

Send a private report to the repository owner with:

- Affected version or commit
- Minimal reproduction steps
- Expected impact
- Suggested fix, if available

## Scope

Lucky Draw Kit is a local draw engine. Reports about draw fairness, predictable seeds, inventory overdraw, or audit tampering risks are in scope.
